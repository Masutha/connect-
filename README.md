# Cnect App Download Page

Universal app download page that automatically redirects users to the appropriate app store based on their device.

## Features

- **Auto-detection**: Automatically detects iOS and Android devices
- **Smart redirect**: iOS users → App Store, Android users → Google Play Store
- **Modern UI**: Clean, responsive design with the Cnect logo
- **Fast**: Redirects in 1.5 seconds with visual feedback

## Links

- **iOS**: [App Store](https://apps.apple.com/za/app/cnect/id6751820432)
- **Android**: [Google Play](https://play.google.com/store/apps/details?id=com.cnect.app)

## Deployment

This is a static HTML page that can be deployed on:

- GitHub Pages (free)
- Netlify
- Vercel
- Any web hosting service

### Deploy to GitHub Pages

1. Push this repository to GitHub
2. Go to Settings → Pages
3. Select the branch (main/master) and root folder
4. Your page will be live at `https://yourusername.github.io/connect-/`

## Usage

Simply share your deployed URL. Users will be automatically redirected to the correct app store.
